Demo with a gridsize of 8x8 pixels, 16x16 pixel hero and tiles, one 24x24 and one 32x32 pixel object.
User arrow keys to move.
Things to do: Use mouse or touch for pathfinding (all objects will block the player and one-directional collision tiles support is kind of buggy). Push objects on each other. Walk or push things into one way directional tiles. Toggle turnbased mode. Explore the Hero properties.
  • Strength - Max mass you can push (most boxes have mass 1, those with a 2 has 2 in mass and the biggest one has 4)
  • MaxCue - Max number of objects you can push
  • Struggle - Slowdown by this factor multiplied with total mass when pushing
  • Velocity - Pixels per second
  • Steps - Number of total steps taken
